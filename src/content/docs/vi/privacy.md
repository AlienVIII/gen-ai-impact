Cập nhật lần cuối: 2026-06-11

Gen AI Impact là static documentation site. Site không có account, comment, form, hoặc user-submitted content.

## Data Collection

Site không chủ động thu thập personal information từ visitor.

Vì site host trên GitHub Pages, GitHub có thể xử lý request metadata tiêu chuẩn như IP address, user agent, timestamp, và requested URL để vận hành, bảo vệ, và duy trì hosting service. Xem privacy policy của GitHub để biết chi tiết về cách họ xử lý dữ liệu.

## Cookies Và Tracking

Site không chủ động set application cookie hoặc chạy custom behavioral tracking.

Nếu sau này thêm analytics, embedded media, comments, newsletter form, hoặc third-party widget, trang này cần được update trước khi publish.

## External Links

Topic có thể link tới external sites. Các site đó có privacy practice riêng và nằm ngoài quyền kiểm soát của site này.

## Drafts Và Source Material

Raw drafts, screenshots, private notes, threads, internal repository names, secrets, và unsanitized logs không được publish. Chỉ sanitized Markdown dưới `src/content/docs/**` được public.
